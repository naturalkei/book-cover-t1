# Issue #97: Keep the v3 leaf visible through its midpoint

## Problem

The CSS leaf rendered only one image with hidden backface visibility. It
therefore disappeared after rotating past 90 degrees, leaving a visible pause
before the deferred target page commit. The gutter cast shadow also extended
opposite the incoming side of the turn.

## Approach

- Render one continuous transform wrapper with front and back image faces.
- Rotate the back face by 180 degrees so it becomes visible after the midpoint
  with correct image orientation.
- Select the target-facing page for single, spread, cover, forward, and
  backward turns.
- Reverse the gutter cast-shadow and ridge direction so forward turns shade
  the left incoming page and backward turns shade the right incoming page.

## Trade-offs

- The CSS fallback remains a rigid continuous plane rather than a true curved
  mesh. The two-sided structure removes the midpoint disappearance while #85
  remains responsible for full curved-paper geometry.
