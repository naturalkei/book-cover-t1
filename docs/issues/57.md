# Issue #57 — Playwright webServer timeout on GitHub Actions

## Symptom

CI failed at the E2E step with:

```
Error: Timed out waiting 60000ms from config.webServer.
```

Vite logged that it was ready in ~309 ms at `http://localhost:5173/`, yet Playwright never detected the server as up.

## Root Cause

`playwright.config.ts` configured:

- **Probe URL**: `http://127.0.0.1:5173` (IPv4)
- **Vite bind**: default `localhost`, which on GitHub Actions Ubuntu runners often resolves to IPv6 `::1` only

Playwright's readiness check hit `127.0.0.1:5173` and got connection refused for the full 60 s timeout even though Vite was listening on `::1:5173`.

## Fix

Added `--host 127.0.0.1` to the `webServer.command` in `playwright.config.ts` so the dev server bind address matches the probe URL.

```ts
command: 'pnpm dev --port=5173 --strictPort --host 127.0.0.1',
url: 'http://127.0.0.1:5173',
```
