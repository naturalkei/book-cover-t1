# Issue #65 — Tighten clsx className line limit to 60

## Problem

The 80-character threshold for splitting Tailwind classes left many long single-line `className` strings, especially in shared bundles and typography utilities.

## Approach

1. Reduced the documented limit from **80 → 60** (`CLASS_LINE_LIMIT` in `src/lib/class-names.ts`).
2. Re-split every clsx argument in `class-names.ts` to ≤ 60 characters.
3. Extracted additional shared bundles (`AppShell`, `LayoutNav`, `GalleryEmptyState`, `ToggleChipLabel`, `PageFlipSurface`, option active/idle states, etc.).
4. Refactored components to use clsx or shared bundles; converted remaining `PageFlip` array-join patterns to clsx.

- `pnpm test` — 181 tests pass
- `pnpm build` — pass
