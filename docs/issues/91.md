# #91 — v3 CSS curl detached mosaic strips

## Problem

The v3 CSS curl fallback split the outgoing page into 12 independently transformed vertical strips. Each strip rotated around its own local pivot and used an independent crop of the page image. During the turn, gaps and crop discontinuities made the page look like a mosaic instead of one sheet of paper.

## Root Cause

- The CSS fallback attempted to approximate a curved mesh with disconnected DOM strips.
- Segment transforms changed orientation without solving the connected position of neighboring strip edges.
- Independent image crops amplified seams when the strips separated.

## Approach

1. Removed the 12-strip DOM renderer from `CssCurlLeaf`.
2. Rendered one continuous outgoing page image.
3. Kept progress-driven motion using the shared curl sample for Y rotation, lift, translation, and shadow.
4. Deferred genuine continuous curvature to the WebGL mesh renderer in #85.
5. Added component and Playwright assertions that the outgoing leaf is one image with no strip children.

## Trade-offs

- The CSS fallback now reads as a coherent sheet but has a lower curvature ceiling.
- Natural local bending remains a WebGL responsibility instead of an inaccurate CSS strip simulation.
