# Issue #6 — feat(gallery): render book gallery at root route

- **Status**: Resolved
- **Milestone**: M1 — Foundation
- **Branch**: `feat/6-gallery-root-route`

## Problem

The root route still rendered the upstream VRT starter `Home` page. The product needed a curated book gallery that links into the reader route.

## Approach

- Added `src/pages/Gallery.tsx` with a hero header, a responsive 1/2/3-column card grid, and an empty state. The page accepts an optional `books` prop so tests can inject fixtures without mocking the module.
- Added `src/components/gallery/BookCard.tsx`: cover image, title, author, and page count chip tinted by the book's `accentColor`. The whole card is a `react-router-dom` `<Link>` with an `aria-label` for screen readers.
- Replaced the existing `Layout` chrome with a dark, slate-950-on-glass shell. Brand swapped from "VRT Starter" to "book-flip-showcase", and the GitHub button is hidden when `VITE_GITHUB_URL` is unset.
- Removed `src/pages/Home.tsx` and switched `App.tsx` to render `Gallery` at `index`. The catch-all `*` route now renders a styled 404.
- Updated `index.html` title / OG metadata and `theme-color` to match the dark identity. Updated `index.css` so `html`/`body` carry a `bg-slate-950` and the scrollbar uses slate-700.
- Wired asset paths through Vite's `BASE_URL`: introduced an `asset()` helper inside `src/data/books.ts` so all `coverSrc` / `pages` URLs are prefixed with `import.meta.env.BASE_URL`. Updated existing data specs to assert path shape rather than absolute prefix.

## Trade-offs

- **Gallery takes an optional `books` prop**: avoids module-mocking in unit tests. Cost: tiny API surface duplication; benefit: simpler, deterministic tests.
- **Dark default in Layout vs theme toggle**: dark default is shipped now; the user-facing toggle waits for M4 (#13). Avoids re-doing chrome in two PRs.
- **`asset()` helper inside `books.ts`**: ties the data module to Vite's `import.meta.env`. Acceptable because this module is web-only by design and the helper falls back to `/` outside Vite.
